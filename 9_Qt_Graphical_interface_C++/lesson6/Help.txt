   Для того, чтобы выбрать файл, из которого необходимо взять
текст, нажмите кнопку "открыть". После этого текст, расположен-
ный в выбранном файле отобразится в диалоговом окне.
   Для сохранения текста, расположенного в диалоговом окне,
нужно нажать кнопку "сохранить", далее выбрать файл, в который 
будет сохранено содержимое диалогового окна.
   Чтобы прочитать содержимое файла без возможности его
изменить нажмите кнопку "прочитать".
   Язык интерфейса можно выбрать, нажам на нужную кнопку
внизу окна.
